# tools-pages

本仓库提供对外说明文档，介绍一个注重隐私与安全的在线工具集合。在线服务地址：https://tools.pycoder.org

## 简介
该项目所有 html 代码采用 AI 生成，设计目标为：
- 每个 html 页面仅提供一个功能
- 数据仅保存在客户端，服务端无任何存储功能，最大程度保障用户隐私
- 可在浏览器中直接使用，没有任何三方依赖

## 使用方法
1. 访问： https://tools.pycoder.org
2. 选择需要的工具并按照界面提示操作
3. 完成后清除浏览器缓存/会话以提高隐私性（如适用）

## 本地部署（简要）
仓库仅包含 html 页面，可直接部署到任意 web 服务中浏览。
